## Run Locally

Clone the project

```bash
  git clone https://github.com/ashabanyan/react-burger.git
```

Go to the project directory

```bash
  cd react-burger
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Build and deploy

```bash
  npm deploy-script
```
